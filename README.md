A minimalist, ultra-premium link-in-bio landing page for **BEACH** (Nightlife/Sauna brand). 

## Features
- **Apple-inspired Aesthetics**: Clean, glassmorphism effects, and fluid animations.
- **Rainbow Theme**: Vibrant, animated gradient background with pastel-tinted buttons.
- **Interactive Pricing**: Accordion-style price list implementation.
- **Mobile First**: Optimized for Instagram/TikTok bio links.

## Technologies
- HTML5
- CSS3 (Variables, Flexbox, Animations, Glassmorphism)
- Vanilla JavaScript

## Setup
1. Clone the repository.
2. Open `index.html` in your browser.
